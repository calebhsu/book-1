# Data Models

Planevent organizes teams based on events and the userIDs associated with each event.


#### Events
- events
  - event name
    - event components
      - component name
        - component description
    - members
      - username
        - last edit time

#### Users
- users
  - username
    - display name
    - email
    - online / offline
